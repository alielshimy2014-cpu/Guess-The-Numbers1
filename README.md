🕹️ Guess The Number Game

A fun and simple number guessing game built with HTML, CSS, and JavaScript!
You can choose your own number range, then try to guess the secret number.
The game gives hints like “Too high!” or “Too low!” until you find the correct answer 🎯

Features:

Custom number range (choose your “from” and “to”)

Colorful red and blue background with yellow text

Responsive design with medium-size buttons

Easy to play and learn from the code
